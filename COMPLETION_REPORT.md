  🎯 USAGE SUMMARY
═════════════════════════════════════════════════════════════════════════════════

BASIC USAGE:
  1. Import useTheme hook
  2. Destructure isDark and toggleTheme
  3. Use in your component
  4. Apply night-* classes to JSX

TAILWIND CLASSES:
  • bg-night-{darker,dark,base,card,hover}
  • text-night-{text,text-secondary,text-subtle}
  • text-night-{green,accent,red,orange}
  • border-night-{border,accent,green,red}
  • shadow-night-{sm,md,lg,glow,glow-blue}

COMPONENT CLASSES:
  • .night-card          → Styled card
  • .night-card-hover    → Card with shimmer
  • .night-product-card  → Product card
  • .night-btn           → Primary button
  • .night-btn-glow      → Button with glow
  • .night-input         → Styled input
  • .night-link          → Styled link
  • .night-badge         → Badge
  • .night-alert         → Alert message
